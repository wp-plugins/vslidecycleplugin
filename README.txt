== Installation ==
1. Upload vslideCycle folder to the /wp-content/plugins/ directory<br>
1. Activate the plugin through the 'Plugins' menu in WordPress<br>
1. You can add a widget to your website or use [vslidecycleplugin] syntax inside pages where you want it to appear. <br>
1. For developers: put `<?php if(function_exists('vslideCyclePlugin')){ vslideCyclePlugin(); } ?>`<br>
1. `<?php do_action('vslideCyclePlugin'); ?>`
