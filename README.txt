== Installation ==
   * Upload vslideCycle folder to the /wp-content/plugins/ directory<br>
   * Activate the plugin through the 'Plugins' menu in WordPress<br>
   * You can add a widget to your website or use [vslidecycleplugin] syntax inside pages where you want it to appear. <br>
   * For developers:  if(function_exists('vslideCyclePlugin')){ vslideCyclePlugin(); } <br>
